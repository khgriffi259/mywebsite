My Work

Unit Number Solver

Several trailers required unit numbers to be installed on the walls individually. It was the processor's job to order the correct quantity of each unit number necessary for each trailer, which was tedious and could get complicated very quickly. Hence, I developed the unit number solver macro. See key features below:

  • Used an event listener to automatically highlight the required characters and quantities required based on the string of characters entered by the user
  • Used a Vlookup to provide the part number of the unit number character
