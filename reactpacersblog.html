React Pacers Blog

This is a React/Redux version of the previous pacers blog project I did for my final project at IUPUI. The technologies used to create this amazing web application are:

  • React
  • Redux
  • Firebase
  • Javascript
  • Materialize

    • Read Blog Posts from Other Users!


    • Dashboard shows activity from all registered users including blog posts and notifications.

    Firebase Authentication


    • Register as a user on the web app
    • Utilizes Firebase Authentication to track registerd users and when a given user is logged in

    Create Blog Posts

    • Once signed up and logged in to the web application, create blog posts to share with other users.

    Firebase Cloud Functions

    • Utilizes firebase cloud functions to deliver real time data.
      • Real time notifications when users sign up
      • Real time notifications when users create blog posts

    Firestore Database

    • Web Application utilizes the firestore database to store blog posts and notifications.
