My Work

Filterable Todo App

This is a full stack todo application (MERN) to help you manage your daily tasks! It has handy filter textbox to help you quickly find specified tasks.

  • React
  • Redux
  • MongoDB
  • Express
  • Node
  • Material-UI

Easily add todos with textbox


  • Clean desktop layout for playing at home!

Filter todos!


  • Just start typing in the textbox located in the navbar and the app will begin to filter your todos!

Delete todos!


  • Simply click on a todo to remove it from the list.
  • A line will appear through the text indicating you are about to delete it.

Todos stored in Database


  • Your todos are stored in MongoDB so they'll still be there if you close the application!
